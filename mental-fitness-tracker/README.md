# 🧠 Mental Fitness Tracker

A comprehensive web application for tracking and analyzing mental health burden across countries using Disability-Adjusted Life Years (DALYs) data.

## 📊 Features

- **Interactive Dashboard**: View mental health metrics for any country
- **Trend Analysis**: Track changes over time with interactive plots
- **Multi-country Comparison**: Compare mental health burden across nations
- **Machine Learning**: Predict DALYs using Random Forest model
- **Global Visualization**: World map showing mental health distribution
- **Data Filtering**: Filter by country and year range
